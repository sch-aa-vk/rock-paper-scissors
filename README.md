# Rock-Paper-Scissors
This is a fun app that helps people make decisions by turning it into a game! It's like the classic 'Rock-Paper-Scissors,' but you're playing against the computer.

> link for the final result: https://sch-aa-vk.github.io/rock-paper-scissors/

Project was developer for the "[INF 201] Software Architecture, Quality, Verification, Validation and Design" university course.

### Project Overview:
PROBLEM: Decision-making can be challenging, and people often find it difficult to choose between options.

SOLUTION: Our project offers a novel approach to decision-making by gamifying the process. Users can play “rock-paper-scissors” with a computer and solve their problems in a quick and entertaining way.
